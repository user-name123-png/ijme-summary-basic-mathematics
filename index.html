ม.ต้น - อสมการและแยกตัวประกอบ

อสมการ แยกตัวประกอบ ข้อสอบแนะนำ โจทย์ปัญหา

🧮 ตอนที่ 1: อสมการ (Inequalities)

1.1 สัญลักษณ์พื้นฐาน

สัญลักษณ์ ความหมาย ตัวอย่าง
> มากกว่า $x > 5$ → x มากกว่า 5
< น้อยกว่า $y < 3$ → y น้อยกว่า 3
≥ มากกว่าหรือเท่ากับ $a ≥ 2$ → a ตั้งแต่ 2 ขึ้นไป
≤ น้อยกว่าหรือเท่ากับ $b ≤ 7$ → b ไม่เกิน 7
≠ ไม่เท่ากับ $c ≠ 4$ → c เป็นอะไรก็ได้ยกเว้น 4

💡 เทคนิคจำ: "ปากจระเข้" — จระเข้หันปากไปทางค่า ที่มากกว่า เสมอ

1.2 การแปลงข้อความ → สัญลักษณ์อสมการ

ข้อความที่พบในโจทย์ สัญลักษณ์
"ไม่เกิน", "สูงสุดไม่เกิน", "อย่างมาก" ≤
"อย่างน้อย", "ไม่ต่ำกว่า", "ตั้งแต่...ขึ้นไป" ≥
"เกิน", "มากกว่า" >
"ไม่ถึง", "น้อยกว่า", "ต่ำกว่า" <
"ไม่เท่ากับ", "คนละค่า" ≠
ตัวอย่าง:

"จำนวนเต็มบวกที่น้อยที่สุดที่มากกว่า 5" → $x > 5$ (ตอบ $x = 6$)

"จำนวนนับที่มากกว่า 3 แต่น้อยกว่า 7" → $3 < x < 7$ (ตอบ $x = 4, 5, 6$)

1.3 กราฟแสดงคำตอบบนเส้นจำนวน

สัญลักษณ์ บนเส้นจำนวน วิธีเขียน
> ⭕ (วงกลมโปร่ง) + แรเงาไปทางขวา $x > a$
< ⭕ (วงกลมโปร่ง) + แรเงาไปทางซ้าย $x < a$
≥ ⚫ (วงกลมทึบ) + แรเงาไปทางขวา $x ≥ a$
≤ ⚫ (วงกลมทึบ) + แรเงาไปทางซ้าย $x ≤ a$

🔹 ข้อควรจำ

  • วงกลมโปร่ง ⭕ = ค่านั้น ไม่รวม ในคำตอบ
  • วงกลมทึบ ⚫ = ค่านั้น รวม ในคำตอบ
  • แรเงา = ค่าทั้งหมดที่ใช้ได้

🔹 การเขียนคำตอบแบบ Interval (สากล)

อสมการ Interval
$x > 2$ $(2, \infty)$
$x ≥ -1$ $[-1, \infty)$
$x < 5$ $(-\infty, 5)$
$x ≤ 3$ $(-\infty, 3]$
$-1 ≤ x < 5$ $[-1, 5)$

1.4 วิธีการแก้อสมการเชิงเส้นตัวแปรเดียว

หลักการสำคัญ: Like สมการ แต่มีข้อแตกต่าง!

ขั้นตอนการแก้

  1. จัดรูป: คูณกระจาย / รวมพจน์ที่คล้ายกัน
  2. ย้ายข้าง: ฝั่งตัวแปรไปซ้าย ฝั่งค่าคงที่ไปขวา (เครื่องหมายเปลี่ยนเมื่อย้ายข้าง)
  3. ⚠️ หัวใจสำคัญ: ถ้า คูณหรือหารด้วยจำนวนลบ → ต้องกลับเครื่องหมายอสมการ!

ตารางสรุปการดำเนินการ

การกระทำ เปลี่ยนเครื่องหมาย?
บวก/ลบ ด้วยจำนวนเดียวกัน ❌ ไม่เปลี่ยน
คูณ/หาร ด้วย จำนวนบวก ❌ ไม่เปลี่ยน
คูณ/หาร ด้วย จำนวนลบ ✅ กลับเครื่องหมาย!
คูณ/หาร ด้วยตัวแปรที่ ไม่รู้เครื่องหมาย ⛔ ห้ามทำ!

ตัวอย่างการแก้

ตัวอย่าง 1: 
3x - 7 > 2x + 5
3x - 2x > 5 + 7    (ย้ายข้าง)
x > 12             ✅
ตัวอย่าง 2: 
-2x + 4 ≤ 10
-2x ≤ 6            (ย้าย 4 ไปขวา)
x ≥ -3             ⚠️ หารด้วย -2 → กลับเครื่องหมาย! ✅

1.5 โจทย์ปัญหาอสมการ — ตัวอย่างข้อที่ 1

โจทย์: อายุของแดงมากกว่า 5 เท่าของอายุของดำอยู่ไม่เกิน 20 ปี ถ้าดำอายุ 7 ปี จงหาอายุของแดงที่เป็นไปได้

วิธีทำ

1
ขั้น 1 — กำหนดตัวแปร: ให้ $R$ แทนอายุของแดง (ปี)
2
ขั้น 2 — สร้างอสมการ: 
อายุของดำ = 7 ปี → 5 เท่า = $5 \times 7 = 35$
"มากกว่า...อยู่ไม่เกิน 20" → $R - 35 ≤ 20$
3
ขั้น 3 — แก้อสมการ: 
$$R - 35 ≤ 20$$
$$R ≤ 55$$
4
ขั้น 4 — ตรวจสอบความเป็นจริง: อายุของแดง > 35 (เพราะ "มากกว่า" 5 เท่าของดำ) ดังนั้น $35 < R ≤ 55$

ตอบ: อายุของแดงมากกว่า 35 ปี แต่ไม่เกิน 55 ปี ✅

1.6 โจทย์ปัญหาอสมการ — ตัวอย่างข้อที่ 2

โจทย์: ส้มมีเงิน 200 บาท ต้องการซื้อดินสอราคาแท่งละ 12 บาท และสมุดราคาเล่มละ 25 บาท โดยดินสอต้องซื้ออย่างน้อย 5 แท่ง และต้องการซื้อสมุดอย่างน้อย 2 เล่ม จงหาจำนวนดินสอและสมุดที่ซื้อได้ทั้งหมดที่เป็นไปได้ (ไม่ต้องซื้อให้หมดเงิน)

วิธีทำ

1
ขั้น 1 — กำหนดตัวแปร: ให้ $x$ แทนจำนวนดินสอ (แท่ง), $y$ แทนจำนวนสมุด (เล่ม)
2
ขั้น 2 — สร้างอสมการเงื่อนไข:
เงื่อนไข อสมการ
เงินไม่เกิน 200 บาท $12x + 25y ≤ 200$
ดินสออย่างน้อย 5 แท่ง $x ≥ 5$
สมุดอย่างน้อย 2 เล่ม $y ≥ 2$
3
ขั้น 3 — หาคำตอบ:
$y$ $12x ≤ 200 - 25y$ $x$ ที่เป็นไปได้
2 $12x ≤ 150$ → $x ≤ 12.5$ → $x ≤ 12$ $x = 5, 6, 7, 8, 9, 10, 11, 12$
3 $12x ≤ 125$ → $x ≤ 10.41$ → $x ≤ 10$ $x = 5, 6, 7, 8, 9, 10$
4 $12x ≤ 100$ → $x ≤ 8.33$ → $x ≤ 8$ $x = 5, 6, 7, 8$
5 $12x ≤ 75$ → $x ≤ 6.25$ → $x ≤ 6$ $x = 5, 6$
6 $12x ≤ 50$ → $x ≤ 4.16$ → $x ≤ 4$ เป็นไปไม่ได้ (ต้อง $x ≥ 5$)

ตอบ: คู่อันดับ $(x, y)$ ที่เป็นไปได้

(5,2), (6,2), (7,2), (8,2), (9,2), (10,2), (11,2), (12,2),
(5,3), (6,3), (7,3), (8,3), (9,3), (10,3),
(5,4), (6,4), (7,4), (8,4),
(5,5), (6,5)

รวมทั้งหมด 20 วิธี ✅

🧮 ตอนที่ 2: การแยกตัวประกอบ (Factoring)

2.1 🔥 สูตรที่ต้องใช้บ่อย

ชื่อ สูตร จำยังไง
ผลต่างกำลังสอง $a^2 - b^2 = (a - b)(a + b)$ "สองหน้า ต่างกัน → (ลบ)(บวก)
กำลังสองสมบูรณ์ (บวก) $a^2 + 2ab + b^2 = (a + b)^2$ "หน้า ² + 2×หน้า×หลัง + หลัง ²"
กำลังสองสมบูรณ์ (ลบ) $a^2 - 2ab + b^2 = (a - b)^2$ "หน้า ² − 2×หน้า×หลัง + หลัง ²"
ผลต่างกำลังสาม $a^3 - b^3 = (a - b)(a^2 + ab + b^2)$ "ลบ → (ก้อนลบ)(สองหน้า + หน้าหลัง + หลัง²)
ผลบวกกำลังสาม $a^3 + b^3 = (a + b)(a^2 - ab + b^2)$ "บวก → (ก้อนบวก)(สองหน้า − หน้าหลัง + หลัง²)

2.2 🧠 เทคนิคจำสูตรกำลังสาม

หลักการจำของทั้งสองสูตร

  • เหมือนกันตรง: วงเล็บหลัง = $a^2 \mp ab + b^2$ (ตัวกลาง ตรงข้ามกับวงเล็บหน้า)
  • ต่างกันตรง: วงเล็บหน้าเป็น $(a \pm b)$ ตามเครื่องหมายเดิม

จำแบบ "Soap" (สำหรับกำลังสาม)

Soap = Same sign   →  (a + b)   หรือ   (a - b)
      Opposite sign   →  -ab  (ตรงข้ามกับวงเล็บหน้า)
      Always Positive →  +b²  (บวกเสมอ!)
สูตร วงเล็บ 1 วงเล็บ 2 (Soap)
$a^3 + b^3$ $(a + b)$ $(a^2 - ab + b^2)$
$a^3 - b^3$ $(a - b)$ $(a^2 + ab + b^2)$

2.3 ตัวอย่างการแยกตัวประกอบแต่ละสูตร

✅ ผลต่างกำลังสอง

ตัวอย่าง: $x^2 - 25$
$x^2 - 25 = x^2 - 5^2 = (x - 5)(x + 5)$

ตัวอย่าง: $4x^2 - 9$
$4x^2 - 9 = (2x)^2 - 3^2 = (2x - 3)(2x + 3)$

✅ กำลังสองสมบูรณ์

ตัวอย่าง: $x^2 + 6x + 9$
$x^2 + 6x + 9 = x^2 + 2\cdot x \cdot 3 + 3^2 = (x + 3)^2$

ตัวอย่าง: $4x^2 - 12x + 9$
$4x^2 - 12x + 9 = (2x)^2 - 2\cdot 2x \cdot 3 + 3^2 = (2x - 3)^2$

✅ ผลต่างกำลังสาม

ตัวอย่าง: $x^3 - 27$
$x^3 - 27 = x^3 - 3^3 = (x - 3)(x^2 + 3x + 9)$

✅ ผลบวกกำลังสาม

ตัวอย่าง: $x^3 + 8$
$x^3 + 8 = x^3 + 2^3 = (x + 2)(x^2 - 2x + 4)$

🧪 ข้อสอบอัตนัยข้อที่ 1: แยกตัวประกอบโดยใช้สูตรผลบวกกำลังสาม (ระดับยาก)

โจทย์: จงแยกตัวประกอบของ $8x^3 + 27y^3$

💡 วิธีคิด

1
ขั้น 1 — สังเกตให้ออกว่าเป็น "ผลบวกกำลังสาม": 
$8x^3 + 27y^3$
2
ขั้น 2 — จัดให้อยู่ในรูป $a^3 + b^3$:
พจน์ จัดรูป
$8x^3$ $= (2x)^3$
$27y^3$ $= (3y)^3$

ดังนั้น $a = 2x$, $b = 3y$

3
ขั้น 3 — ใช้สูตร $a^3 + b^3 = (a + b)(a^2 - ab + b^2)$: 
$8x^3 + 27y^3 = (2x + 3y)[(2x)^2 - (2x)(3y) + (3y)^2]$
4
ขั้น 4 — คำนวณในวงเล็บหลัง: 
$(2x)^2 = 4x^2$
$(2x)(3y) = 6xy$
$(3y)^2 = 9y^2$
5
ขั้น 5 — ได้คำตอบสุดท้าย: 
$$\boxed{8x^3 + 27y^3 = (2x + 3y)(4x^2 - 6xy + 9y^2)}$$

✅ ตรวจคำตอบ (คูณกลับ)

ทดสอบด้วยการแทนค่า $x = 1, y = 1$:

ฝั่งซ้าย : $8(1)^3 + 27(1)^3 = 8 + 27 = 35$
ฝั่งขวา : $(2 + 3)(4 - 6 + 9) = 5 \times 7 = 35$ ✅

🧪 ข้อสอบอัตนัยข้อที่ 2: โจทย์ปัญหาอสมการแนวประยุกต์ (หาค่าเฉลี่ย)

โจทย์: คะแนนสอบ 4 ครั้งแรกของสมชายคือ 72, 68, 85, และ 79 คะแนน ตามลำดับ ถ้าสมชายต้องการให้คะแนนเฉลี่ย 5 ครั้ง อย่างน้อย 80 คะแนน ในการสอบครั้งที่ 5 เขาจะต้องได้คะแนนอย่างน้อยกี่คะแนน

💡 วิธีคิด

1
ขั้น 1 — กำหนดตัวแปร: ให้ $x$ แทนคะแนนสอบครั้งที่ 5
2
ขั้น 2 — สร้างอสมการจากเงื่อนไข "ค่าเฉลี่ย ≥ 80": 
$$\frac{72 + 68 + 85 + 79 + x}{5} ≥ 80$$
3
ขั้น 3 — แก้อสมการ: 
$$\frac{304 + x}{5} ≥ 80$$
คูณทั้งสองข้างด้วย 5 (บวก → ไม่กลับเครื่องหมาย):
$$304 + x ≥ 400$$
ย้ายข้าง:
$$x ≥ 400 - 304$$
$$x ≥ 96$$

✅ คำตอบ

สมชายต้องได้คะแนนสอบครั้งที่ 5 อย่างน้อย 96 คะแนน 🔥

🧪 ตรวจสอบความสมเหตุสมผล

กรณี คะแนนครั้งที่ 5 ค่าเฉลี่ย ผ่าน
ต่ำสุดที่ผ่าน 96 $(304+96)/5 = 80.0$ ✅
ต่ำกว่า 95 $(304+95)/5 = 79.8$ ❌
สูงกว่า 100 $(304+100)/5 = 80.8$ ✅

เรื่อง ข้อควรจำ
อสมการ คูณ/หารด้วยลบ → กลับเครื่องหมาย!
ผลต่างกำลังสอง $a^2 - b^2 = (a-b)(a+b)$ → "ลบแล้วคู่"
กำลังสองสมบูรณ์ ตรวจ $2ab$ ว่าเท่ากับพจน์กลางหรือไม่
กำลังสาม ใช้ Soap: Same, Opposite, Always Positive
โจทย์ปัญหา กำหนดตัวแปร → สร้างอสมการ → แก้ → ตรวจสอบ
